Methods

(static) mixinRemoval(param0)

Mixes a callback into the element that is called whenever element.remove() is called (this happens e.g. when an element is deleted from the diagram). If the mixin is applied to the same element multiple times, all the supplied the callbacks are executed in the order the respective mixins have been called.

param0 object Options object.
