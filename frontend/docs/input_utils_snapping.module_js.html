(static) snapAlongAxisToCoordinates(fromPosition, toCoordinates, axis, tolerance) → {Object}

Snaps the coordinate of fromPosition given in 'axis' to the closest of the positions given in toCoordinates, if the distance is less than the tolerance value. Returns a new position.
Parameters:
Name Type Description
fromPosition Object The position to snap.
toCoordinates Array.<{x: number, y: number}> The positions to snap to.
axis string The snapping direction ('x' or 'y').
tolerance number The maximum distance between coordinates for snapping.
