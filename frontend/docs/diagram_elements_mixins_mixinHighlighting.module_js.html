(static) mixinHighlighting(param0)

Mixes a boolean 'highlighted' state into the given element. The state can be read with element.isHighlighted(), changed with element.highlight() and element.unhighlight(). Side effects such as graphical representation of the highlighted state are not added with this mixin.
Parameters:

param0 object Options object.
