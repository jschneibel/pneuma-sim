(static) mixinPosition(param0)

Mixes a {{ x: number, y: number }} position into the given element. The position can be read with element.getPosition() and changed with element.setPosition(). Changing the intial position object directly won't have an effect on the position of the element. Side effects such as graphical representation of the position are not added with this mixin.

param0 object Options object.
